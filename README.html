@manzhixing/utilsxy ​

前端常见的操作和功能的函数

Install ​

bash
npm install @manzhixing/utilsxy

# OR

yarn add @manzhixing/utilsxy

Usage ​

ts
// 全量引入
import * as utilsxy from '@manzhixing/utilsxy';
utilsxy.typeOf('chengbotao'); // "string"

// 差量使用
import { typeOf } from '@manzhixing/utilsxy';
typeOf('chengbotao'); // "string"
